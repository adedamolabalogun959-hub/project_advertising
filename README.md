# project_advertising
Linear regression model built with statsmodel.api using Ordinary Least Square.
Exploratory Data Analysis , handling missing values.
Regression model assumption validated using Linearity, Normality of Residuals, Homoscedasticity with diagnostic plots(Residuals vs Fitted plot), (Normal Q-Q plot), (Scale-Location).
Correlation matrix of variables. 
Interpreting R-squared, Co-efficients, P-values.
ROI- based Recommendations for marketing budget allocation from predictions.
